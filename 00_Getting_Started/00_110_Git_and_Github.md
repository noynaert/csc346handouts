# 00.110 Git and Github

Git and Github are essential tools for modern software development. They allow you to manage your code, collaborate with others, and keep track of changes over time.

## Git

Git is a version control system. It allows you to keep track of changes to your code over time. It also allows you to collaborate with others on the same code base.

Git runs on your local computer. You can use Git from the command line or from a graphical user interface (GUI). Most modern IDEs have built-in support for Git.

## Github

GitHub is a web-based platform that uses Git for version control. It allows you to host your code repositories online, collaborate with others, and manage projects. There are other similar platforms, such as GitLab and Bitbucket.

## Practical approach

Git and Github can be confusing. Git was developed as a command line tool for maintaining the Linux kernel. Kernel programmers are not normal human beings, so the user interface is not very user friendly. Github was developed later as a web-based platform for hosting Git repositories.
